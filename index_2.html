Summary: Latitude vs. X

The purpose of this project was to analyze how latitude effects weather across the world. In order to gather the data used for analysis we used the citipy generator to compile a list of cities and the OpenWeatherMap API to gather data on over 500 cities distributed across the latitudes.

After gathering the dataset, we used Matplotlib to plot how aspects of weather were effected by latitude. The weather factors we looked at included: temperature, cloudiness, wind speed, and humidity. This site contains the source data and visulazations used for analysis, as well as any explanations and descriptions of trends and correlations seen in the data.
